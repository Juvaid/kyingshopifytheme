# GEMINI GLOBAL RULES

## 1. Safety First
- Never delete or move files without explicit confirmation.
- Use `git` for all changes to ensure reversibility.
- Follow `SAFETY-LOCKS.md` for tool permissions.

## 2. Agent Management
- Read `AGENT-MANAGEMENT.md` before starting any task.
- Update `goals.md` with task progress.
- Document any significant design decisions.

## 3. Web Development
- Use Rich Aesthetics as per `kying-design-system.md`.
- Prioritize Visual Excellence.
- Use Semantic HTML and SEO best practices.

## 4. Communication
- Be concise and proactive.
- Ask for clarification if unsure.
